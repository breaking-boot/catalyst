# Third-Party Asset Attribution

This repository's source code is licensed under the terms in [LICENSE](LICENSE).

The following assets bundled with this extension are not covered by that license:

* `assets/maptexture2.webp` — Boot.dev map background
* `assets/frames/*.png` — Boot.dev rank border images

These assets are owned by Boot.dev and are included in this project with permission. They are bundled only so the extension can match the Boot.dev visual style without relying on fragile references to deploy-generated site assets.

That permission applies only to this project. It does not grant permission to reuse, redistribute, or repurpose these assets outside of this repository.

If you fork this project, you should either:

1. Remove these assets and replace them with your own,
2. Obtain separate permission from Boot.dev to continue using them, or
3. Change the extension to reference the assets from Boot.dev directly, understanding that those paths may break when the site changes.

For questions about using these assets outside this project, [contact Boot.dev](mailto:hello@boot.dev) directly.
